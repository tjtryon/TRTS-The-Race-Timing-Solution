📘 The Race Timing System for Cross Country and Road Races – Documentation

Welcome to the 🏁 The Race Timing System for Cross Country and Road Races. This platform lets you manage, time, and display race results for cross country and road race events.

⚙️ Console Features

  • RFID scanning via file watcher 🏷️
  • Manual finish entry ⌨️
  • Millisecond timing ⏱️
  • Team scoring 🥇

🌐 Flask Web Features

  • View results online 🌍
  • Edit race results 🔧
  • View usage notes 📚 and documentation 📑

🔧 Installation

python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
cd console
python3 race_timing_console.py

📂 Console Directory Structure

console/
├── data/  🗄️ Race databases
├── web/   🌐 Flask app
├── race_timing_console.py ⚙️ Console CLI
└── requirements.txt 📜

🛠️ Troubleshooting

  • Flask app won't start: Check your Python version and virtual environment.
  • Login fails: Confirm config.db has an admin user.
  • RFID scans missing: Verify the RFID input file is being updated.
  • No sound: Ensure beep.mp3 is present in the directory.
